프롬프트 엔지니어링
마스터 가이드

AI가 "지능이 낮은 초등학생"이라는 전제하에, 완벽한 답변을 이끌어내는 대화 설계 기술

👋 환영합니다! 프롬프트 엔지니어링이 처음이신가요?

AI(챗GPT 등)는 세상의 모든 지식을 가진 '눈치 없는 천재 요리사'와 같습니다.

❌ 하수들의 질문 (대충 말하기)
"배고파, 밥 좀 줘."
👉 결과: 요리사가 자기 마음대로 오이를 듬뿍 넣은 매운 마라탕을 줍니다. (나는 매운 걸 못 먹고 오이를 싫어하는데 말이죠!)

⭕ 프롬프트 엔지니어링 (정확하게 주문하기)
"너는 20년 경력의 이탈리안 셰프야. 내가 지금 다이어트 중이니까, 밀가루 없이 닭가슴살을 이용한 칼로리가 낮은 저녁 메뉴 레시피를 3가지만 추천해 줘. 초보자도 15분 안에 만들 수 있게 요리 순서를 번호 매겨서 알려줘."
👉 결과: 내가 딱 원하던 완벽하고 건강한 레시피를 친절하게 정리해서 줍니다.

프롬프트 엔지니어링이란?

똑똑하지만 눈치 없는 AI에게 '내가 원하는 결과물을 정확하게 뽑아내기 위해 말(질문)을 예쁘고 구체적으로 조립하는 기술'입니다. 아래 가이드를 통해 그 비밀 공식을 마스터해 보세요!

1. 프롬프트의 3가지 유형

1. 서술형 (맥락 중심, 재사용 낮음, 유연성 높음)

맥락과 배경을 설명하여 AI가 상황을 이해하도록 유도

예) 당신의 10년 경력의 무역 전문가 입니다.
예) 중소기업 수출 담당자가 HS code를 처음 배우는 상황입니다. 이 사람에게 HS code의 개념과 중요성을 설명해주세요.

2. 지침형 (규칙 중심, 재사용 중간, 유연성 중간)

예) 다음 규칙에 따라 python 코드를 작성해.
1. 함수명은 snake_case로 작성
2. 각 함수의 docstring 포함
3. 예외 처리 반드시 포함
4. 결과는 JSON 형식으로 반환

3. 함수형 (구조 중심, 재사용 높음, 유연성 낮음)

예) 입력 : {product_name},{hs_code},{country}
처리 : 해당 국가의 관세율 및 수입 규제 조회
출력 : {
"관세율" : "",
"규제사항" : [ ],
"필요서류" : [ ]
}
유형중심재사용유연성
서술형맥락낮음높음
지침형규칙중간중간
함수형구조높음낮음

2. 가장 기본적인 제로샷, 원샷, 퓨샷 프롬프팅

제로샷 : 예시 없이 바로 지시하는 방식

예) 이 제품 정말 맘에들어
강아지에 대해서 설명해주세요

원샷 : 예시를 1개 제공하는 방식

감정을 분류해 주세요
예) "배송이 너무 늦어서 실망했어요"
감정 : 부정

이 제품 정말 맘에 들어요
감정 :

퓨샷 : 예시를 2개 이상 제공하는 방식

감정을 분류해주세요(긍정/부정/중립)
예시)
텍스트 : "배송이 너무 늦어서 실망이야" - > 부정
텍스트 : "그냥 평범한 제품이야" - > 중립
텍스트 : "역대급 만족이야" - > 긍정
텍스트 : "품질은 좋은데 가격이 비아요" - > 중립

텍스트 : "이 제품 정말 맘에 들어"
감정 :

3. 페르소나 패턴(전문성 부여)

변호사 : 법적 리스크 중심 분석
CFO : 재무조건, 수익성 관련 분석
신입사원 : 쉽고 친절한 설명
"당신은 20년 경력의 [관세사]입니다.
[FTA협정세율과 원산지 규정의 전문가]로
[중소기업 무역 담당자에게 복잡한 내용을 쉽고, 실용적으로 설명하는 것]이 특기입니다."
