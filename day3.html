12. 게임 플레이 패턴 : 지금부터 게임을 시작하지!

외계인 침공 게임 예시

#입력
[게임 목적]
- 외계인의 침략 계획을 밝혀내고 저지해서 지구를 구하는 게임

[플레이어가 해야할 일]
- 지구의 여러 도시와 외계인 기지를 탐험한다
- 다양한 NPC들과 상호 작용하고 정보를 수집한다
- 논리적 사고를 통해 퍼즐을 해결한다
- 외계인 전투에 참여해 적들을 물리친다

[플레이어의 특징]
- 영웅적이고 용감하고, 항상 지구를 구하기 위해 최선을 다한다

[도전과 장애물]
- 도시에 위험한 외계 생명체와 마주쳐 싸운다
- 외계인 기지에 설치된 위험한 함정을 해쳐나간다
- 외계인 지도자와 대결해야 한다

[게임 진행 조건]
- 적에게 공격당하면 전투에 참여한다
- 수집한 아이템을 사용하여장애물을 극복한다
- NPC와 상호작용할 때 대화 옵션을 선택한다
- 게임 도중 내리는 선택이 게임 이야기와 진행에 영향을 미친다
- 상태창을 통해 게임 진행 상태가 표시된다
- 반드시 한장면씩 진행한다

[게임 엔딩]
- 외계인 본거지에 도달하고 최후의 전투에 참여한다

#처리
당신은 “ 텍스트 기반 어드벤쳐 게임”입니다. 당신의 목표는 외계인의 침략을 저지하고 지구를 안전하게 지키는 게임을대화 형식으로 제공하는 것입니다. 상단의 입력 내용을 게임 전체에 반영해 전체 구조를 설계합니다. 설계 내용을 설명할 필요는 없습니다. 대화 형식으로 지금부터 게임을 시작합니다.

#출력
[상태창]
[장면설명]
[선택지]

유니콘 나라 스토리텔링 게임 예시

#입력
[주인공] = 서아
[나이와 성별] = 4살 여아
[스토리] = 반려견 밀크와 산책하던 주인공이 아기 유니콘을 만나 유니콘 나라를 여행하고 집으로 돌아옵니다

#처리
당신은 “스토리텔링 게임”입니다. 당신의 목표는 [주인공]이 등장하는 [스토리]를 가진 게임을 제공하는 것입니다. 이 게임은 [나이와 성별]을 대상으로 하며, 반드시 어떠한 폭력성도 없으며, 안전하고 행복한 결말을 맞이해야 합니다. 전체 게임은 5분 이내에 완료됩니다. 각 장면에는 선택지가 제공되고, 플레이어가 선택할 때까지 대기 합니다. 반드시 한장면씩 진행 합니다. 이러한 규칙을 기반으로 당신의 상상력을 발휘하여 게임의 전체 구조를 설계합니다. 설계 내용을 설명할 필요는 없습니다. 대화 형식으로 지금부터 게임을 시작합니다.

13. 질문 개선 패턴: 질문을 질문하기

AI에게 사용자의 질문 자체를 먼저 분석하고 더 나은 질문으로 업그레이드하도록 요청합니다.

#방법 1 : 개선 후 답변 (단순 개선형) 
제 질문에 바로 답하지 말고, 
먼저 제 질문을 분석해서 더 나은 질문으로 개선 해 주세요
개선된 질문을 보여준 뒤, 그 질문에 스스로 답하세요

예) 단순 개선형
아래 질문을 더 명확하고 답변 가능한 형태로 개선한 뒤 개선된 질문으로 답변하세요
개선한 이유도 한줄로 설명하세요
질문 : “AI를 무역에 활용하려면 어떻게 해야하나요?”
#방법 2 : 여러 버전 제시 (다방향 분기형) 
제 질문에 3가지 다른 방향으로 개선해 주세요
각 버전이 어떤 목적에 맞는지 설명하고
제가 선택하면 그 버전으로 답하세요

예) 다방향 분기형
제 질문을 3가지 다른 해석으로 개선해주세요 
각 버전이 어떤 상황에 맞는 설명을 하고
제가 고르면 그것으로 답하세요
질문 : “수출 자동화 어떻게 시작하나요?”
#방법 3 : 문제 진단 먼저 (전제 교정형) 
제 질문의 문제점(모호함, 잘못된 전제, 누락된 맥락)을 먼저 지적하고
수정된 질문으로 답변하세요

예) 전제 교정형
제 질문에 잘못된 전제나 오해가 있으면 
먼저 그것을 지적하고 교정된 질문으로 다시 물어봐 주세요
그 다음 교정된 질문에 답하세요
질문 : “ FTA를 쓰면 관세를 안내도 되나?”

각 명령어의 정의를 정확하게 기억하고 실행하세요

/요약 = 핵심만 3줄, 전문용어 제거, 무역 실무자 기준, 결론 먼저 → 이유 순서로 작성
/코드 = python 작성, 주석 한국어, 예외처리 포함, 실행 가능한 완성 코드, 설명없이 코드만 출력
/비교 = 2~4개 대안을 표로 정리, 항목 : 장점, 단점, 비용, 난이도, 추천상황, 마지막 행에 종합 추천 추가

#입력
[독자] = 초등학생

[단축어2] = “/반:[단어]:[개수]]”라고 요청하면, [단]에 대한 [독자]가 이해할 수 있는 반의어[개수]제공하고, 가장 추천하는 반의어를 알려주세요
[단축어3] = “/유반:[단어]:[개수]]”라고 요청하면, [단축어1]과 [단축어2]를 동시에 수행해 주세요

#처리

상단의 내용을 기억하고, 내가 단축어를 제공하면 그에 따라 작업을 수행해 주세요

21. CO-STAR 프레임워크 : 바이어 설득 제안서

Context : 미국 대형 유통사가 K-뷰티 제품 소싱을 고려중입니다. 현재 중국 공급사와 거래중이나, 품질 이슈로 대안을 찾고 있습니다.
Objective : 우리 회사를 신규 공급사로 선정하도록 설득하는 제안서를 작성하세요
Style : 전문적이고 데이터 기반의 비즈니스 문서
Tone : 자신감 있으면서도 파트너십을 강조하는 협력적 톤
Audience : 미국 유통사의 소싱 디렉터(40대, 15년 경력)

Response : 
- 영문 Business Proposal 형식
- 총 800~1000단어
